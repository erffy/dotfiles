## Erffy Dotfiles
- My config files.

### Installation
```sh
# clone repository
git clone https://github.com/erffy/dotfiles.git
# install required packages
cd dotfiles && pacman -S --needed - < packages
# copy repository contents to config
cd .. && cp dotfiles/* $HOME/.config/
```
